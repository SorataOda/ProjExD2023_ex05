# ゲーム のタイトル：スピードワゴン
## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
主人公のキャラクターがスクロールするステージでジャンプ入力（スペースキー）で障害物を避けたり、アイテムを取ったりします！

###共通基本機能
* 主人公キャラクターに関するクラス
*　背景の機能
*　コインクラス
